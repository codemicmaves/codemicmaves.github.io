Lisp Constructs

In this page, we list some commonly used emacs-lisp constructs when customizing the emacs configuration, adding functionality or writing plugins.

Variables

Use a backtick instead of a single quote for quoting (`) and prefix all symbols to be evaluated with a comma (,) to evaluate symbols in a quoted list. 

(add-to-list 'list `(symbol ,evaluated_var))

Use the let function to initialize variables for a set of statements. 

(let ((var1 value1)
      (var2 value2))
  (statement_1)
(statement_2))

Write cons cells explicitly, as (car . cdr), if the first item has a special meaning (for readability). 

(setq modules '(module . (feature_1 feature_2 feature_3)))

Lists

Use car to get the first item, cdr to get remaining and nth to get the nth. 

(car '(1 2 3 4))   # => 1
(cdr '(1 2 3 4))   # => (2 3 4)
(nth 2 '(1 2 3 4)) # => 3

Control statements

Use when or unless to do things only for one branch of a control statement. 

(when (condition)
  (statement_when_true))

(unless (condition)
  (statement_when_false))

But, there is always the regular if, if you need. 

(if (condition)
    (statement_when_true)
  (statement_when_false))

More conditions? No problem, use cond. 

(cond (is-linux-p do_linux_work)
      (is-mac-p   do_mac_work)
      (is-win-p   install_linux_then_do_work))

Looping statements

Use dolist to loop through a list and process something one item at a time 

(dolist (item list)
  (body_statements_using item))

If you need indices, use dotimes instead. 

(dotimes (index count)
  (body_statements_using index))

Want both index and item like python's enumerate? 

(dolist (pair (loop for i from 0 for var in list collect (cons i var)))
  (let ((index (car pair))
        (item  (cdr pair)))
    (do_something_with index item)))

Emacisms

Use the interactive function if you need to call the function with M-x. 

(defun superfun ()
  "This is a super fun function."
  (interactive)
  (do_super_fun_things))

Want the full path to the currently loading el file? or the directory? 

(message load-file-name)
(message (file-name-directory load-file-name))

Recursively add subdirectories to load-path. 

(let (default-directory "root-dir/")
  (normal-top-level-add-subdirs-to-load-path))

Elispisms

Namespace with a prefix/symbol format. 

(defun my/superfun ()
  (this_is_my_superfun_not_yours))

Double dash (--) function names to indicate internal functions to others. 

(defun acme/superfun--in-joke ()
  (lol_funny))
